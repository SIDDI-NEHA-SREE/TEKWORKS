# Titanic Survival Prediction System

Deep Learning Based Passenger Survival Prediction using Artificial Neural Networks.

## Features

- Passenger Class input
- Age input
- Fare input
- Survival prediction
- Confidence score
- Probability visualization
- TensorFlow ANN deployment
- Streamlit interface

## Model Architecture

Input Layer: 3 neurons
Hidden Layer: 2 neurons
Output Layer: 1 neuron
Activation: Sigmoid

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
streamlit run app.py
```

## Deployment

Deploy using Streamlit Community Cloud.
